# Check if a number is Palindrome or Not

# Problem Statement:  
Given an integer N, return true if it is a palindrome number otherwise return false.

A palindrome number is a number that reads the same backward as forward.

# Examples:
```js
Example 1:  
Input: N = 4554  
Output: Palindrome Number  
Explanation: The reverse of 4554 is 4554, so it is a palindrome.

Example 2:  
Input: N = 7789  
Output: Not Palindrome  
Explanation: The reverse of 7789 is 9877, so it is not a palindrome.
```

// Approach
To check if a number is a palindrome, we reverse the number and compare it with the original number.
If both are equal, the number is a palindrome.

// Algorithm
1. Initialize revNum = 0.
2. Store the original number in dup.
3. While n > 0:
   - Get last digit using n % 10.
   - Update revNum = revNum * 10 + last digit.
   - Remove last digit using n = n / 10.
4. Compare revNum with dup.
5. If equal return true, else return false.

# Code
```js
import java.util.*;

class Solution {
    public boolean palindrome(int n) {
        int revNum = 0;
        int dup = n;

        while (n > 0) {
            int ld = n % 10;
            revNum = (revNum * 10) + ld;
            n = n / 10;
        }

        return dup == revNum;
    }
}

public class Main {
    public static void main(String[] args) {
        int number = 4554;
        Solution obj = new Solution();

        if (obj.palindrome(number)) {
            System.out.println(number + " is a palindrome.");
        } else {
            System.out.println(number + " is not a palindrome.");
        }
    }
}
```

# Time Complexity
O(log10N + 1), where N is the number of digits in the number.

 Space Complexity
O(1), only constant extra space is used.
